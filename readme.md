# osx-screensaver

> Start the screensaver on OS X systems.


## Install

```
$ npm install --save osx-screensaver
```


## Usage

```js
const osxScreensaver = require('osx-screensaver');

osxScreensaver().then(() => {
	console.log('Screensaver started');
});
```
